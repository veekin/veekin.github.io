七、常用命令总结

命令 说明
hexo new "title" 创建新文章
hexo clean 清除缓存
hexo g 或 hexo generate 生成静态页面
hexo s 或 hexo server 启动本地预览
hexo d 或 hexo deploy 部署到远程

八、可能的问题与解决方案

1. 权限问题：Permission denied

建议使用 HTTPS 链接而非 SSH 链接（如 https://github.com/...）

确认 hexo-cli 是否全局安装，并检查 npm 环境变量。
